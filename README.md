# Libraries
## print.hpp
A library for printing the elements of various containers. Useful for debugging.
### Supported Containers
- std::vector (Multidimensional)
- std::map
- std::unordered_map
- std::set
- std::unordered_set

## container_traits.hpp
A container version of the library <type_traits>. Used for deducing the type of container passed onto a template. Could be useful in making other template libraries.
### Supported Containers
- std::vector
- std::map
- std::unordered_map
- std::set
- std::unordered_set
- std::list
- std::stack
- std::deque
- std::deque
