## Path library
The Path library aims to simplify and improve the process of performing operations on file systems with the `<filesystem>` library.

## Minimum C++ Version
- `C++17`

## Includes
- `<iostream>`
- `<vector>`
- `<fstream>`
- `<filesystem>`
### Windows
- `<windows.h>`
### Linux
- `<unistd.h>`
- `<sys/stat.h>`
